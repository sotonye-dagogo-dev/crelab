5. Motion Tokens

Easings and durations that define Crelab's interaction language.

Token Name Value Description
--ease-out cubic-bezier(0.16, 1, 0.3, 1) Decelerating — natural settling, modal entrances
--ease-in-out cubic-bezier(0.45, 0, 0.55, 1) Symmetric — page transitions, element swaps
--ease-spring cubic-bezier(0.34, 1.56, 0.64, 1) Spring-like overshoot — checkmarks, celebratory reveals
--duration-fast 150ms Hover states, micro-interactions, toggles
--duration-base 250ms Standard transitions: dropdowns, modals, filter changes
--duration-slow 400ms Page transitions, section reveals, large element changes
--duration-slower 600ms Hero entrances, staggered reveals, confirmation screens

6. Breakpoints

Mobile-first responsive architecture. Design is optimised for brand marketing managers browsing between meetings.

Mobile <640px
Tablet 640–1024px
Desktop >1024px

Mobile

Viewport: < 640px

Grid: 2-column masonry

Layout: Single column, sticky bottom CTA on profiles, full-screen bottom sheets for booking flow

Nav: Hamburger → full-screen overlay

Tablet

Viewport: 640 – 1024px

Grid: 3-column masonry

Layout: Expanded navigation visible, side-by-side service packages, full-width hero on profiles

Nav: Full nav bar visible

Desktop

Viewport: > 1024px

Grid: 4–5 column masonry

Layout: Full navigation, sticky sidebar booking widget on profile pages (320px), right-side booking sheet (480px)

Nav: Full nav with CTAs right-aligned
